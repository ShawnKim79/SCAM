# SACM 자기 진단 분석 프롬프트

이 프롬프트는 SACM(Self-Analysis Cognitive Model) 자기 진단 결과를 분석하는 데 사용할 수 있습니다. 아래 프롬프트를 복사하여 AI 어시스턴트에게 전달하면, 여러분의 진단 결과에 대한 맞춤형 분석을 받을 수 있습니다.

## 기본 분석 프롬프트

```
나는 SACM(Self-Analysis Cognitive Model) 자기 진단 테스트를 완료했습니다. 아래는 내 점수입니다:

1. 입력 처리 스타일: [점수]/15
   - 1.1 신호 민감도: [점수]
   - 1.2 논리-감정 신호 비율: [점수]
   - 1.3 감정 조절 방식: [점수]

2. 인지 모듈 진단: [점수]/20
   - 2.1 문제 접근 방식: [점수]
   - 2.2 패턴 인식 및 적용: [점수]
   - 2.3 감정적 과부하 대응: [점수]
   - 2.4 메타인지 수준: [점수]

3. 출력 생성 방식: [점수]/15
   - 3.1 의사소통 스타일: [점수]
   - 3.2 응답 속도: [점수]
   - 3.3 감정-논리 균형 표현: [점수]

4. 학습 및 사회화 패턴: [점수]/10
   - 4.1 경험 후 성찰: [점수]
   - 4.2 피드백 수용도: [점수]

5. 오류 처리 및 회복: [점수]/15
   - 5.1 과부하 감지 능력: [점수]
   - 5.2 회복 탄력성: [점수]
   - 5.3 오류 허용도: [점수]

6. 자기 발전 패턴: [점수]/15
   - 6.1 경험 기반 패턴 생성: [점수]
   - 6.2 실패 패턴 자동 수정: [점수]
   - 6.3 유연성 정도: [점수]

총점: [점수]/90

이 결과를 바탕으로 내 인지-감정 프로필에 대한 분석과 이해를 돕고, 개선할 수 있는 영역과 활용할 수 있는 강점을 알려주세요.
```

## 심층 분석 프롬프트 (상황별 분석)

```
나는 SACM 자기 진단 테스트를 완료했습니다. 아래는 내 점수입니다:

[기본 분석 프롬프트와 동일한 점수 정보 입력]

추가로, 내가 가장 자주 경험하는 어려운 상황은 [구체적인 상황 설명]입니다. 
이런 상황에서 내 SACM 프로필이 어떻게 영향을 미치는지, 
어떤 강점을 활용할 수 있고 어떤 영역을 개선하면 더 효과적으로 대처할 수 있는지 분석해주세요.
```

## 비교 분석 프롬프트 (과거 대비 현재)

```
나는 SACM 자기 진단 테스트를 두 번 완료했습니다. 아래는 이전과 현재의 점수입니다:

[이전 날짜] 결과:
[기본 분석 프롬프트와 동일한 점수 정보 입력]

[현재 날짜] 결과:
[기본 분석 프롬프트와 동일한 점수 정보 입력]

두 결과를 비교하여 어떤 변화가 있었는지, 어떤 영역이 발전했고 어떤 영역이 변화했는지 분석해주세요. 
이러한 변화가 내 일상생활과 의사결정에 어떤 영향을 미칠 수 있는지도 설명해주세요.
```

```
어떤 강점을 더 활용할 수 있고, 어떤 영역을 개발하면 더 효과적으로 일할 수 있을지 분석해주세요.
```

## 관계 최적화 분석 프롬프트

```
나는 SACM 자기 진단 테스트를 완료했습니다. 아래는 내 점수입니다:

[기본 분석 프롬프트와 동일한 점수 정보 입력]

내 주변 중요한 사람들의 특성은 다음과 같습니다:
- [사람 1]: [간략한 특성 설명]
- [사람 2]: [간략한 특성 설명]
- [사람 3]: [간략한 특성 설명]

내 SACM 프로필을 고려할 때, 이들과의 관계에서 어떤 강점과 도전이 있을 수 있는지, 
그리고 어떻게 더 효과적으로 소통하고 관계를 발전시킬 수 있는지 분석해주세요.
```

## 발전 계획 프롬프트

```
나는 SACM 자기 진단 테스트를 완료했습니다. 아래는 내 점수입니다:

[기본 분석 프롬프트와 동일한 점수 정보 입력]

향후 6개월 동안 내 인지-감정 시스템을 발전시키기 위한 구체적인 계획을 수립하고 싶습니다. 
내 SACM 프로필을 고려하여, 가장 큰 영향을 미칠 수 있는 2-3개 영역에 초점을 맞춘 
단계별 발전 계획을 제안해주세요. 각 단계마다 실행 가능한 구체적인 활동과 
진행 상황을 측정할 수 있는 방법도 포함해주세요.
```
